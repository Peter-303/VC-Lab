# 第二章 设计原则

> 掌握设计的四大基本原则，学会运用它们提升设计质量

## 2.1 对比原则

对比是设计中最强大的原则之一，它通过差异来吸引注意力，创造视觉层次。

### 对比的类型

| 类型 | 说明 | 效果 |
|------|------|------|
| **大小对比** | 元素尺寸的差异 | 突出重要元素 |
| **颜色对比** | 明暗、冷暖的差异 | 增强可读性 |
| **形状对比** | 几何形状的差异 | 增加视觉趣味 |
| **空间对比** | 疏密、留白的差异 | 引导视觉流向 |

### 对比的应用技巧

- 使用大字号突出标题，小字号展示正文
- 深色背景配浅色文字，或反之
- 重要按钮使用醒目的颜色

## 2.2 对齐原则

对齐确保设计元素在视觉上保持一致的排列，建立视觉秩序。

### 对齐方式

**左对齐**：最常用的对齐方式，适合长文本阅读

**右对齐**：创造独特的视觉效果，适合强调内容

**居中对齐**：适用于标题和重要信息

### 对齐的重要性

- 📖 提高可读性
- ✨ 增强专业感
- 🧩 建立视觉秩序

## 2.3 重复原则

重复通过重复使用相同或相似的元素来创造一致性和统一性。

### 重复的应用

- **颜色重复**：使用统一的配色方案
- **形状重复**：重复使用相似的图形元素
- **排版重复**：保持一致的字体和大小
- **间距重复**：保持统一的留白和间距

## 2.4 亲密性原则

亲密性原则将相关的元素组合在一起，不相关的元素分开，建立清晰的信息层级。

### 亲密性的好处

- 📁 组织信息结构
- 🧹 减少视觉混乱
- ⚡ 提高信息传达效率

## 2.5 实践练习

**练习任务**：选择一个你喜欢的网站或App界面，分析它是如何运用四大设计原则的。

**记录要点**：
1. 哪里使用了对比？
2. 元素是如何对齐的？
3. 哪些元素被重复使用？
4. 信息是如何分组的？
